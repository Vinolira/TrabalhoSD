É necessário executar o arquivo Conversão_ANSI_UTF-8_ECMA-94.py pelo terminal de comando a partir da pasta em que o arquivo estiver contido.
O programa emite a seguinte mensagem: "nome do arquivo: "
O usuário deve informar a localização do arquivo que deseja converter (endereço relativo ou absoluto, caso o arquivo não esteja na mesma pasta que o código).
O arquivo deve ser um .txt codificado em ANSI, do contrário podem haver problemas na execução.
